# Tuisheng Style (蜕升学院)

Dark scholarly aesthetic fusing ancient manuscript textures with cyberpunk energy. High information density, academic rigor, cognitive impact.

## Element Combination

```yaml
canvas:
  ratio: portrait-3-4
  grid: triple-panel-stack # 3 horizontal panels stacked vertically

image_effects:
  texture: aged-parchment-dark # dark paper with subtle grain, foxing, vignette
  filter: high-contrast
  vignette: heavy-corners # darkened corners for depth

typography:
  decorated: stroke-text | neon
  tags: black-white | ribbon
  direction: horizontal
  bilingual: true # Chinese primary + English in parentheses for technical terms
  math: latex-style # proper mathematical typesetting

decorations:
  emphasis: red-vertical-bar | callout-box-red-border
  background: dark-textured-parchment
  doodles: geometric-constructions | faint-astronomical-charts
  frames: none
```

## Visual Elements

### Background Treatment (CRITICAL)

- **Dark aged parchment**: NOT flat black — organic, textured, slightly mottled dark surface
- **Subtle grain**: Fine paper/canvas texture visible at full resolution
- **Corner vignette**: All four corners darken further toward near-black
- **Foxing/patina**: Very subtle lighter speckles and stain marks (inverted from light parchment — here they are slightly lighter patches on dark)
- **Faint background marks**: Ghost-like geometric constructions, astronomical chart fragments, compass roses in ~10-15% opacity — like viewing the infographic on an ancient dark manuscript

### Three-Panel Structure

- Each image contains **3 stacked horizontal panels**
- Each panel is a self-contained section with its own title + visual + text
- Panels separated by subtle horizontal division (shade shift, thin gold line, or gap)
- Creates a "scrolling knowledge card" feel

### Diagrams & Visuals

- **Wire-frame 3D**: Technical illustration style, warm off-white lines on dark
- **Flow charts**: Connected nodes with gold/teal arrows
- **Data visualizations**: Charts, heat maps, comparison bars
- **Mathematical formulas**: LaTeX-quality rendering, cream/gold color
- **Maps/Spatial**: When relevant, rendered in the dark palette with territory color-coding

### Callout Boxes

- Thin gold or teal border (2px)
- Slightly lighter dark fill (#252530)
- Contains key insight / core argument
- Red vertical bar (4px) on left side for blockquote-style emphasis points

## Typography

### Heading Hierarchy

| Level | Role               | Style                                         | Color                         |
| ----- | ------------------ | --------------------------------------------- | ----------------------------- |
| H1    | Panel title        | Bold serif (宋体/明体), very large (~36-42pt) | Gold #D4A520 or cream #E8E0D0 |
| H2    | Sub-section        | Bold serif, ~20-24pt                          | Cream #E8E0D0 or teal #40B8B8 |
| H3    | Body text          | Regular serif, ~14-16pt, line-height 1.6-1.8x | Warm gray #B8A898             |
| H4    | Labels/annotations | Smaller serif, ~11-13pt                       | Muted #A89888                 |
| Math  | Formulas           | LaTeX Computer Modern style                   | Gold #D4A520                  |

### Title Format

- Pattern: `主题词：副标题说明` (Topic: Subtitle)
- Colon separator is consistent
- Main topic in gold, subtitle in cream

### Bilingual Convention

- Primary: Chinese (Simplified)
- Secondary: English in parentheses for technical terms
- Pattern: `中文术语 (English Term)`
- Creates scholarly, international credibility

### Special Text Treatments

- **Gold vertical bar**: Thick gold line left of key points (blockquote indicator)
- **Red highlight**: Key warning terms or critical points
- **Teal highlight**: Technical terms or positive emphasis
- **Numbered lists**: `1.` `2.` `3.` with hanging indent
- **Bullet points**: `•` (filled circle) in gold

## Information Density

### Per Panel Target

- 1 main concept title
- 1 primary visual element (diagram, chart, or illustration)
- 3-6 bullet points or text blocks
- 0-2 mathematical formulas or data points
- 0-1 callout/highlight box with key insight
- ~80-150 Chinese characters + formulas per panel

### Balance

- 40% visual elements, 40% text, 20% whitespace
- Dense but organized — grid structure prevents chaos
- Generous line spacing within text blocks

## Best For

- Deep analysis / cognitive content (认知升维)
- Technical explanations with visual support
- Historical/geopolitical analysis
- Framework and model breakdowns
- High-value knowledge cards
- Academic-meets-street-smart content

## DO

- Maintain the three-panel structure consistently
- Use bilingual labels for technical terms
- Include at least one diagram/visual per panel
- Keep the aged dark texture — never flat/clean
- Use gold for primary emphasis, teal for secondary, red for warnings
- Make each panel self-contained but part of a coherent narrative

## DON'T

- Use bright/pastel colors — everything should feel deep and rich
- Use cute/kawaii decorations — this is serious intellectual content
- Leave panels text-only — always include a visual element
- Use flat solid backgrounds — texture is essential
- Overcrowd — dense ≠ cluttered, maintain reading flow
