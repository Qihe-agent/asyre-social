# 启合涌现 (Emergence Studio) AI Image Generation Style

## Color Palette
- Primary: Professional blue (#2563EB)
- Secondary: Warm gold (#D69E2E)
- Background: Clean light (#F7F8FC) or pure white
- Text: Deep navy (#1A1A2E)
- Overall: Corporate, clean, trustworthy

## Visual Language
- Clean geometric layouts, structured compositions
- Abstract shapes for data visualization (not literal charts)
- Minimal, professional — think McKinsey meets Notion
- Flat or isometric illustrations, no photorealistic
- Gradient overlays (blue to indigo) for hero sections

## Prompt Constraints
Always include in prompts:
- "clean minimal design"
- "professional corporate style"
- "blue and gold color scheme"
- "white or light background"
- "NO watermarks, NO borders"

Never include:
- "dark", "cyber", "grunge", "aggressive", "cute"
- "neon", "gothic", "hand-drawn", "messy"

## Per-Image Type
| Type | Additional Direction |
|------|---------------------|
| Cover | Clean geometric composition, bold title space, abstract blue gradient |
| Data/Stats | Isometric data visualization, clean grid layout, number-forward |
| Process/Flow | Connected nodes, flowchart aesthetic, structured progression |
| Comparison | Side-by-side clean panels, structured table-like layout |
| Ending | Minimal, logo-forward, single blue gradient element |
